# Dataflow Modelling

Projects implemented using dataflow modeling — described using continuous
assignment (`assign`) statements and Verilog operators, rather than
instantiating individual logic gates or using behavioral constructs.

## Projects

- **2x1_Multiplexer** — 2-to-1 multiplexer
- **4x1_Multiplexer** — 4-to-1 multiplexer

Each project folder contains its own README with the specific problem, design
approach, and result, along with the Verilog source, testbench, schematic image,
and waveform image.
